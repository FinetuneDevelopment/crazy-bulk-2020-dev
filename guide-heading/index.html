Heading shadow

The modifying class h-shadow can be added to any heading (or heading class) to give the outline shadow effect below. Note that the content of the heading must be replicated in a data-clone attribute, in order for the effect to render correctly.

Because of the nature of the shadows, headings using this class cannot wrap, or they become illegible. This means they should be used with care and special attention should be paid to how they behave on mobile.
