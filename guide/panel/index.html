Text repeated behind an image

Using dynamically-created SVGs lets text repeat behind an image, where the text is always the full width of the container.

Note that if the text within the SVG changes, then the viewBox and the y values within the SVG need to be adjusted to accomodate for the length of the text string. The CSS will remain the same, however.

Note that the SVG will remain locked to the bottom of its container, and will maintain a square aspect ratio, unless the viewBox is adjusted to make it match its parent.
