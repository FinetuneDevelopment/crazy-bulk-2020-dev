Flags

Flags are short sentences or single words which protrude into panels in various ways.

The flag elements themselves are agnostic about position and can be used as inline elements, if required. In order to achieve the layouts below, a punch class should be added to the container and a positioning utility class should be added to the flag element. For example:

  • pos-ne - element is positioned to the top right
  • pos-sw - element is positioned to the bottom left

... and so on.
