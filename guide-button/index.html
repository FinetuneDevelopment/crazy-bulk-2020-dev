Tertiary call-to-action buttons

These were originally designed to be how secondary call-to-action buttons looked, when placed against dark backgrouds, but unfortunately, limitations with inheritance and CSS prevent this from being possible.

Note that because the text colour of the buttons is white and the background colour of the button is transparent, care should be taken to only use this element in a context where it has sufficient contrast.
